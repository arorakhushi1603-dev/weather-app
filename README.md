# Weather App 

A responsive weather application built using JavaScript and an external API. This app allows users to search for any city and view real-time weather details like temperature, humidity, and weather conditions.



## Features

* Search weather by city name
* Displays temperature, humidity, and weather conditions
* Real-time API data fetching
* Clean and responsive user interface



## Tech Stack

* HTML
* CSS
* JavaScript
* Weather API

##  Project Structure

* `index.html` – Main structure of the app
* `style.css` – Styling of the application
* `script.js` – Handles API calls and logic



##  How to Run Locally

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
